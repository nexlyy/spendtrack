# Политика конфиденциальности SpendTrack

_Последнее обновление: 4 июля 2026 г._

SpendTrack — приложение для личного учёта расходов, которое работает **полностью
на вашем устройстве** и офлайн.

## Какие данные мы собираем

**Никакие.** Приложение не собирает, не передаёт и не хранит ваши данные на
серверах. У приложения нет доступа к интернету (разрешение INTERNET не
запрашивается), нет аналитики, рекламы и трекеров.

## Где хранятся ваши данные

Все записи о тратах, доходах, бюджетах, целях и настройки хранятся **локально
на вашем устройстве** (в базе данных приложения, IndexedDB). Они не покидают
устройство и доступны только вам.

## Резервные копии

Функция «Сохранить копию» создаёт файл с вашими данными и отдаёт его в системный
диалог сохранения/«Поделиться» — вы сами выбираете, куда его положить (в память
устройства, облако и т.д.). Приложение не отправляет этот файл никуда
автоматически. Автоматический облачный бэкоп Android для приложения отключён,
чтобы данные не покидали устройство без вашего ведома.

## Импорт банковских выписок

Если вы импортируете CSV-выписку, файл читается и разбирается **локально на
устройстве**. Он никуда не отправляется.

## Разрешения

Приложение не запрашивает разрешений на доступ к личным данным, местоположению,
контактам, камере и т.п. Выбор файла для импорта/восстановления идёт через
системный диалог и не даёт приложению постоянного доступа к вашим файлам.

## Дети

Приложение не предназначено специально для детей и не собирает данные о ком-либо.
